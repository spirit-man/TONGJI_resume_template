## 使用说明
1. 字体设置：fonts中可以替换为其他字体
2. 在需要调整节间间距的地方使用`\vspace{数值}`，如：
```latex
\section{教育经历}
\vspace{0.5cm}  % 可以根据需要调整数值
```
